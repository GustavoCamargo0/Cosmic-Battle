# 🚀 Cosmic Battle

## 📖 Sobre o Projeto

**Cosmic Battle** é um jogo inspirado nos clássicos arcades de invasão espacial, especialmente *Space Invaders*, trazendo uma identidade visual própria e mecânicas simples, mas divertidas.

O projeto foi desenvolvido utilizando apenas tecnologias web nativas, com foco no aprendizado de lógica de programação, manipulação do DOM, eventos de teclado e desenvolvimento de jogos em JavaScript puro.

---

## ✨ Funcionalidades

* 🚀 Controle da nave do jogador.
* 👾 Sistema de inimigos com movimentação automática.
* 🔫 Disparo de projéteis.
* 💥 Detecção de colisões.
* 🎯 Sistema de eliminação de inimigos.
* 📱 Estrutura adaptável para diferentes tamanhos de tela.
* 🎨 Interface inspirada em jogos clássicos de arcade.

---
